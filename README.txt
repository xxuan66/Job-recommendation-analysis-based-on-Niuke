# 项目描述：
基于Neo4j构建职位技能需求的知识图谱，对不同职位之间的技能关联性进行深度分析。通过知识图谱的可视化展示，用户能够直观地了解各职位之间的技能关系，为用户的职业发展提供科学的数据支持和决策依据。

# 项目职责：
（1）使用Selenium对牛客网的职位数据进行自动化爬取，并对职位描述和岗位要求进行数据清洗与预处理。基于TF-IDF和Word2Vec相结合的模型，精准计算职位需求与岗位职责的文本相似度，实现了职位的精准搜索推荐功能。系统还为用户生成推荐职位描述的词云图，帮助用户快速了解职位的核心要求。（2）基于Neo4j构建职位技能需求的知识图谱，对不同职位之间的技能关联性进行深度分析。通过知识图谱的可视化展示，用户能够直观地了解各职位之间的技能关系，为用户的职业发展提供科学的数据支持和决策依据。
